2. 핵심 동작 구조

브라우저가 자바스크립트를 처리하는 3단계 과정

1

HTML 문서 작성

<script> 태그 내부에 코드를 삽입합니다.

2

브라우저 로드

브라우저가 HTML을 읽다 태그를 만나면 해석을 시작합니다.

3

DOM 제어

HTML 요소(DOM)를 조작하여 화면을 동적으로 변경합니다.

3. 간단한 예제

가장 기본적인 방법은 HTML 파일 내부에 직접 작성하는 것입니다. 아래 코드를 복사하여 HTML 파일로 저장한 뒤 브라우저로 열어보세요.

  • <script> 태그: 코드의 시작과 끝을 알립니다.
  • console.log(): 개발자 도구 콘솔에 내용을 출력합니다.
  • alert(): 사용자 화면에 경고창을 띄웁니다. 
<!DOCTYPE html>
<html>
<head>
    <title>JS Setup Example</title>
</head>
<body>
    <h1>안녕하세요!</h1>
    
    <script>
        // 이 부분이 자바스크립트 코드입니다.
        console.log("Hello, JavaScript!");
        alert("자바스크립트가 설정되었습니다.");
    </script>
</body>
</html>

4. 실행 및 디버깅 방법

브라우저 개발자 도구 활용

  1. 1. Chrome 브라우저에서 HTML 파일을 엽니다.
  2. 2. F12 또는 Cmd + Opt + I를 누릅니다.
  3. 3. Console 탭에서 실행 결과를 확인합니다.
