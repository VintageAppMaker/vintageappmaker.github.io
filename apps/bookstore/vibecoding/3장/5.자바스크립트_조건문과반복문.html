Flow Control in JS

자바스크립트의 핵심인 조건문과 반복문을 통해
컴퓨터에게 판단력을 부여하고 효율적인 작업을 지시하는 법을 배웁니다.

01

조건문 (Conditional Statements)

필요한 이유

프로그램은 항상 유연해야 합니다. 사용자의 입력이나 시스템의 상태와 같은 특정한 조건에 따라 다른 결과를 도출하기 위해 사용합니다.

시작
점수 >= 70?
NO
YES
불합격 메시지
합격 메시지

Example: if-else

const score = 71;

if (score >= 70) {
  // 조건이 참(true)일 때 실행
  console.log("합격입니다."); 
} else {
  // 조건이 거짓(false)일 때 실행
  console.log("불합격입니다.");
}

코드 해설

  • • if (조건)은 괄호 안의 논리가 참인지 판별합니다.
  • • 현재 score가 71이므로 조건문은 참이 되어 첫 번째 블록이 실행됩니다.
02

반복문 (Loops)

노가다 대신하기

수백, 수천 개의 데이터를 일일이 코드로 작성하는 것은 불가능에 가깝습니다. 반복문은 단 몇 줄의 코드로 대량의 연산을 자동화합니다.

종류 사용 대상 특징
for 숫자, 고정 횟수 정확한 반복 횟수 제어 가능
while 조건 기반 반복 조건이 참인 동안 끝없이 반복
for-of 배열 (Array) 배열 요소를 하나씩 순회

Basic for loop

// 0부터 2까지 반복 (3회)
for (let i = 0; i < 3; i++) {
  console.log("현재 번호:", i);
}

// 출력 결과:
// 현재 번호: 0
// 현재 번호: 1
// 현재 번호: 2

for-of loop

const fruits = ["사과", "바나나", "포도"];

for (const fruit of fruits) {
  console.log("과일 이름:", fruit);
}

// 출력 결과:
// 과일 이름: 사과
// 과일 이름: 바나나
// 과일 이름: 포도
Expert Tip

강사의 팁: 컴퓨팅 사고로 바라보기

바이브 코딩(Vibe Coding) 관점에서 조건문과 반복문을 잘 쓰려면 "문제를 쪼개는 능력(분해)"이 중요합니다.

1 반복 (Iteration)

"리스트에 있는 모든 항목에 대해 동일한 박스를 그려라."
→ for-of 활용

2 조건 (Condition)

"단, 품절된 항목은 회색으로 보여라."
→ if 활용

이처럼 로직을 말로 먼저 설계한 뒤 코드로 옮기면 훨씬 쉽게 프로그램을 만들 수 있습니다.
