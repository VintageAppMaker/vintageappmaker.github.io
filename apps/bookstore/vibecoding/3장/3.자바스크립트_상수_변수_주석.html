Programming Essentials

프로그래밍의 가장 기본이 되는 세 가지 기초 개념:
변수, 상수, 그리고 주석을 쉽고 명확하게 알아봅니다.

Variable

1. 변수 (let, var)

프로그램을 만들 때 이름, 나이, 점수와 같이 언제든지 변할 수 있는 데이터를 저장하고 관리하기 위해 필요합니다. 데이터를 담아두는 '이름표가 붙은 상자'라고 생각하면 쉽습니다.

age
상자 선언
25
초기값 대입
30
값 교체 가능
javascript example
let age = 25; // 상자에 25 저장
age = 30; // 값을 30으로 변경
console.log(age); // 결과: 30

* 참고: 매뉴얼(11p)에 따르면 현재는 오류 방지를 위해 let 사용을 강력히 권장합니다.

Constant

2. 상수 (const)

절대로 변해서는 안 되는 고정된 값(예: 생년월일, 수학의 파이값 등)을 보관합니다. 실수로 값을 바꾸는 것을 방지해주는 '잠긴 상자'입니다.

LOCK
BIRTHDAY = "1990-01-01"
CHANGE
BIRTHDAY = "2000-01-01"
❌ Error!
const BDAY = "1990-01-01";
// BDAY = "2000-01-01";
TypeError: Assignment to constant...
Comment

3. 주석 (Comment)

코드는 컴퓨터가 읽는 것이지만, 주석은 사람이 읽기 위해 작성합니다. 나중에 내가 혹은 동료가 코드를 이해하기 쉽게 도와주는 '코드 속의 메모장'입니다. 컴퓨터는 이 부분을 완벽하게 무시합니다.

//

한 줄 주석

코드 옆이나 위에 간단한 메모를 할 때 사용

/* */

여러 줄 주석

기능이 복잡할 때 상세한 설명을 작성

// 이 아래는 점수를 저장하는 변수입니다
let score = 100;

/*
 * 여러 줄의 설명이 필요할 때
 * 이렇게 작성할 수 있습니다.
 */

Tip: 매뉴얼(82p)에서는 /** ... */ 형태의 JSDoc을 사용하여 더 체계적인 정보를 기록하는 법을 권장합니다.

핵심 요약

  • 01 변수(let): 변할 수 있는 값, 상자 속 내용 교체 가능
  • 02 상수(const): 변하지 않는 고정값, 자물쇠가 걸린 상자
  • 03 주석(//): 사람을 위한 메모, 컴퓨터는 무시
