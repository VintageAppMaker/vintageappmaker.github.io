Beginner's Guide

왜 자바스크립트를
배워야 하는가?

현대 소프트웨어 서비스의 중심인 웹(Web).
그 중심을 관통하는 가장 강력한 언어, 자바스크립트의 세계로 초대합니다.

웹의 필수 언어

전 세계 모든 웹 브라우저에서 실행되는 유일한 언어입니다. 구글, 네이버, 유튜브 등 우리가 매일 사용하는 모든 웹 서비스는 자바스크립트 없이는 작동할 수 없습니다.

풀스택 개발

화면(Front-end)부터 서버(Back-end)까지 언어 하나로 정복. 개발 효율의 끝판왕입니다.

PC

데스크탑 앱

VS Code, Discord처럼 강력한 PC 프로그램을 만듭니다.

App

모바일 앱

아이폰과 안드로이드 앱을 동시에 개발할 수 있습니다.

Srv

서버 개발

Node.js를 통해 초고속 데이터를 처리하는 서버를 구축합니다.

자바스크립트의 생태계

JavaScript
웹 브라우저 React, Vue, Angular
모바일 앱 React Native, Ionic
서버 개발 Node.js, Nest.js

HTML과 JS의 관계

  • 1

    HTML (뼈대): 웹페이지의 구조를 정의하는 정적인 상태

  • 2

    JavaScript (근육): 생명력을 불어넣어 클릭, 애니메이션 등 동적인 제어 담당

"자바스크립트가 없다면 웹은 그저 움직이지 않는 신문에 불과합니다."

백문이 불여일견: 직접 확인하기

example.js 
<!-- HTML -->
<h1 id="message">안녕하세요!</h1>
<button onclick="changeText()">클릭하세요</button>

// JavaScript
function changeText() {
  // 1. 요소를 찾아서
  const element = document.getElementById('message');
  
  // 2. 내용을 변경합니다.
  element.innerText = '자바스크립트로 웹이 움직입니다!';
}

안녕하세요!

실제 자바스크립트가 작동 중입니다.

1

상호작용의 마법

document.getElementById는 마치 주소를 찾는 것과 같습니다. 수많은 HTML 요소 중 우리가 제어할 대상을 정확히 짚어냅니다.

2

실시간 변화

innerText를 통해 새로고침 없이 화면의 글자나 스타일을 바꿉니다. 이것이 현대적인 웹 애플리케이션(SPA)의 핵심 원리입니다.

!

결론

자바스크립트는 단순한 코딩 언어를 넘어, 사용자와 세상이 소통하게 만드는 연결고리입니다. 개발자로 성장하고 싶다면 가장 먼저 손에 넣어야 할 무기입니다.
