HTML 제어와 브라우저의 이해

웹페이지를 살아 움직이게 만드는 마법, 자바스크립트를 활용한 HTML 수정 기법과 개발자 도구 활용법을 알아봅니다.

1. 왜 HTML을 제어해야 할까요?

정적인 HTML 문서는 한 번 로드되면 변하지 않습니다. 하지만 사용자의 클릭, 데이터 업데이트 등에 맞춰 화면을 바꾸기 위해서는 **자바스크립트가 실시간으로 HTML을 수정**해야 합니다.

핵심 개념 (DOM): 브라우저는 HTML 문서를 읽어 '나무(Tree)' 형태의 지도인 DOM(Document Object Model)을 만듭니다.

HTML 제어 흐름 도식

Step 1 HTML 문서 작성
2
Step 2 DOM 생성 (브라우저가 지도를 만듦)
3
Step 3 JavaScript가 DOM을 통해 제어

2. 실시간 수정 예제 체험

자바스크립트가 어떻게 HTML의 내용을 바꾸는지 직접 확인해 보세요.

script.js 
// 1. 수정할 대상을 ID로 찾습니다.
const el = document.getElementById("demo");

// 2. 글자 내용을 바꿉니다.
el.textContent = "내용이 성공적으로 변경되었습니다!";

// 3. 스타일도 함께 바꿀 수 있습니다.
el.style.color = "#3b82f6";
el.style.fontWeight = "bold";

3. 핵심 요약: 선택하고 바꾸기

① 요소 선택하기 (찾기)

  • ●
    getElementById("ID")

    가장 빠르고 정확하게 특정 요소 하나를 찾을 때 사용

  • ●
    querySelector(".클래스")

    CSS 선택자 문법을 사용하여 유연하게 찾을 때 사용

② 내용 수정하기 (바꾸기)

  • ●
    .textContent

    태그 안의 순수한 글자 내용만 변경

  • ●
    .innerHTML

    새로운 HTML 태그를 포함하여 구조 자체를 변경

  • ●
    .setAttribute()

    이미지 경로(src)나 링크(href) 같은 속성을 변경

4. 크롬 개발자 도구 (실시간 연습장)

01.

도구 열기

웹브라우저에서 F12 또는 Ctrl+Shift+I를 눌러 개발자 도구를 엽니다.

요소 선택

상단 Elements(요소) 패널을 누른 뒤, 수정하고 싶은 코드 줄을 찾습니다.

03.

즉시 수정
