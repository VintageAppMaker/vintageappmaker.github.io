01. 기본 골격

<html>, <head>, <body>

브라우저에게 웹 문서임을 알리고, 정보와 실제 내용을 구분하는 약속입니다.

  • html: 전체 문서를 감싸는 그릇
  • head: 제목, 설정 등을 담는 '뇌'
  • body: 실제 보이는 '몸통'
<html> <head><title>내 첫 페이지</title></head> <body>내용이 보입니다.</body> </html>
02. 제목

<h1> ~ <h6>

정보의 중요도를 나타내며 검색 엔진 최적화의 기준이 됩니다.

<h1>메인 제목</h1> <h2>서브 섹션 제목</h2>
H1 H2 H3 H4
03. 본문

<p>

텍스트를 문단 단위로 나누어 가독성을 높입니다.

<p id="myPara"> 안녕하세요. 반갑습니다! </p>
"안녕하세요. 반갑습니다."
04. 레이아웃

<div>

여러 요소를 하나로 묶어 디자인과 레이아웃을 잡는 필수 태그입니다.

<div id="outer" style="..."> 상자 안의 내용 </div>
05. 연결

<a>

다른 페이지로 이동하는 하이퍼링크를 생성합니다.

구글로 이동
실제링크
06. 목록

<ul>, <li>

항목을 나열하며 동적 리스트 생성 시 가장 많이 쓰입니다.

<ul id="list"> <li>첫 번째 항목</li> <li>두 번째 항목</li> </ul>
  • 항목 1
  • 항목 2
07. 입력

<input>

사용자로부터 정보를 입력받는 창입니다. type에 따라 변신합니다.

<input type="text" placeholder="입력..." />
08. 행동

<button>

클릭 이벤트를 발생시켜 자바스크립트 함수를 실행합니다.

<button id="btn"> 클릭하세요 </button>
09. 양식

<form>

입력창과 버튼들을 묶어 서버로 데이터를 전송하는 봉투 역할을 합니다.

  • onsubmit 속성으로 데이터 검증 가능
  • 여러 input 데이터를 그룹화
<form onsubmit="..."> <input type="text" /> <button type="submit">제출</button> </form>
10. 생명력

<script>

HTML에 동작(JS)을 불어넣습니다. 화면 뒤의 로직을 담당합니다.

<script> console.log("Hello!"); </script>
